# 未來魔法學院｜親子共學 AI 賦能

Static website for `https://ai-family-camp.launchdock.app`.

## Local Preview

Open `index.html` directly, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://127.0.0.1:8080`.

## Structure

- `index.html`: one-page activity website
- `styles.css`: visual system and responsive layout
- `assets/`: generated concept images
- `docs/visual-system.md`: reusable creative direction, prompts, and video plan

## Deployment Notes

This site is intentionally dependency-free. It can be deployed as a static site on GitHub Pages, Cloudflare Pages, Netlify, or any static host. The intended custom domain is:

```text
ai-family-camp.launchdock.app
```
